# Fundo Mágico - Gerador de Backgrounds com IA

## Descrição

O Fundo Mágico é uma aplicação web que permite aos usuários gerar backgrounds personalizados para seus sites ou projetos. Utilizando o poder da inteligência artificial, a ferramenta transforma descrições de texto em código HTML e CSS, criando visuais incríveis de forma rápida e fácil.

## Como Funciona

1.  **Descreva o background:** O usuário insere uma descrição do background que deseja criar em uma caixa de texto.
2.  **Gere o código:** Ao clicar no botão "Gerar Background Mágico", a aplicação envia a descrição para uma API de IA.
3.  **Visualize e utilize:** A IA retorna o código HTML e CSS correspondente, que é exibido na tela. O usuário pode então copiar e colar o código em seus próprios projetos.

## Tecnologias Utilizadas

*   **HTML5:** Estrutura da página.
*   **CSS3:** Estilização da interface e dos backgrounds gerados.
*   **JavaScript:** Lógica da aplicação, incluindo a comunicação com a API de IA.
*   **N8N:** Automação de fluxo de trabalho para criar o background personalizado.

## Como usar

1. Clone o repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Descreva o background que você deseja na caixa de texto.
4. Clique em "Gerar Background Mágico".
5. Copie o código HTML e CSS gerado e utilize em seu projeto.
